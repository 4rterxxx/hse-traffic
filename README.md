# Прогнозирование зарплат и классификация IT-разработчиков на основе резюме hh.ru

Проект для автоматического предсказания зарплат и определения уровня разработчиков на основе признаков из резюме с сайта hh.ru.

## Структура проекта

```
hw1/
├── pipeline/                 # Пайплайн обработки данных
│   ├── features.py
│   ├── handlers.py
│   ├── io_utils.py
│   ├── processor.py
│   └── __init__.py
├── hh_regression/             # Модель предсказания зарплат
│   ├── linear_reg.py
│   ├── trainer.py
│   ├── predictor.py
│   └── __init__.py
├── hh_classification/         # Классификация IT-разработчиков
│   ├── config.py
│   ├── loader.py
│   ├── features.py
│   ├── model.py
│   ├── main.py
│   └── __init__.py
├── resources/                 # Сохраненные веса моделей
├── hh_results/                # Результаты классификации
├── app.py                     # Главный интерфейс
├── requirements.txt
└── README.md
```

## Установка

```bash
pip install -r requirements.txt
```

## Использование

### Предсказание зарплат (исходное задание)
```bash
# Обработка данных
python app.py process hh.csv

# Обучение модели
python app.py train x_data.npy y_data.npy

# Предсказание
python app.py predict x_data.npy
```

### Классификация IT-разработчиков
```bash
# Запуск классификации
python app.py classify hh.csv
# или с указанием папки для результатов
python app.py classify hh.csv my_results
```

## Результаты классификации

После выполнения в папке `hh_results/` (или указанной вами):

- `labeled_resumes.csv` - размеченные резюме
- `class_distribution.png` - график распределения классов
- `model_results.png` - матрица ошибок и метрики
- `feature_importance.csv` - важность признаков
