# Лабораторная работа по обработке данных 

Проект для обработки и подготовки данных из CSV файла с использованием паттерна "Цепочка ответственности".

## Установка

1. Клонируйте репозиторий
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Запуcтите

```bash
python app.py path/to/hh.csv
```
