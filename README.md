# Прогнозирование зарплат и классификация уровней разработчиков на основе резюме hh.ru

Проект для автоматического предсказания зарплат и определения уровня (junior/middle/senior) IT-разработчиков на основе признаков из резюме с сайта hh.ru.

## Структура проекта

```
hse-traffic/
├── pipeline/              # Пайплайн обработки данных
│   ├── features.py       # Извлечение признаков
│   ├── handlers.py       # Обработчики (цепочка ответственности)
│   ├── io_utils.py       # Утилиты ввода/вывода
│   ├── labeling.py       # Разметка уровней junior/middle/senior
│   ├── processor.py      # Главный процессор пайплайна
│   └── __init__.py
├── regression/           # ML модель для регрессии (зарплаты)
│   ├── linear_reg.py     # Модель линейной регрессии
│   ├── trainer.py        # Обучение модели
│   ├── predictor.py      # Предсказание зарплат
│   └── __init__.py
├── classification/       # Классификация уровней (задание 3)
│   ├── classifier.py     # Модели классификации
│   ├── trainer.py        # Обучение и оценка
│   ├── predictor.py      # Предсказание уровня
│   ├── visualization.py  # Графики и визуализация
│   └── __init__.py
├── resources/            # Сохраненные веса моделей
├── app.py               # Главный интерфейс приложения
├── requirements.txt     # Зависимости
└── README.md           # Документация
```

## Установка

Установите зависимости:
```bash
pip install -r requirements.txt
```

## Использование

### Задание 1: Пайплайн обработки данных

**Ветка:** `task1-pipeline`

```bash
# Обработка CSV файла с резюме
python app.py process hh.csv
```

**Результат:**
- Создаются файлы `x_data.npy` (признаки) и `y_data.npy` (целевые значения)
- Используется паттерн "Цепочка ответственности" для обработки данных

### Задание 2: Модель предсказания зарплат

**Ветка:** `task2-regression`

```bash
# Обучение модели
python app.py train x_data.npy y_data.npy

# Предсказание зарплат
python app.py predict x_data.npy
```

**Результат:**
- Модель сохраняется в папке `resources/`
- Возвращается список предсказанных зарплат в формате float

### Задание 3: Классификация уровней разработчиков (PoC)

**Новая функциональность:** Определение уровня специалиста (junior/middle/senior) по данным резюме.

```bash
# 1. Подготовка данных с размеченными уровнями
python app.py prepare-class hh.csv

# 2. Обучение классификатора
python app.py train-class x_class.npy y_class.npy

# 3. Предсказание уровня для новых данных
python app.py predict-level x_class.npy

# 4. Сравнение разных моделей классификации
python app.py compare-class x_class.npy y_class.npy
```
